O que é o SULCUS

Uma nova forma
de estudar medicina.

Construído especificamente para estudantes da Universidade Jean Piaget de Angola.

O SULCUS combina resumos gerados por IA, mapas mentais interativos, flashcards com revisão espaçada e um assistente clínico sempre disponível.

Cada conteúdo foi desenhado para o currículo angolano — não é genérico. É medicina como a estudas.

SULCUS
/ ˈsʌl.kəs /
substantivo · anatomia
Sulco ou fissura na superfície do cérebro, separando os giros cerebrais. Cada sulco aumenta a área cortical disponível para processamento.
// Plural: sulci · Do latim sulcus (sulco, ranhura) · Estrutura que define a inteligência pela superfície, não pelo volume.
